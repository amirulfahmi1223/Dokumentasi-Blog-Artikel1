Untuk kesekian kalinya sobat lagi- lagi saya kembali memposting tutorial ataupun coding dari bahasa pemrograman C++ sobat maklum sobat pemrograman terstruktur C++ ini sobat merupakan mata kuliah saya juga di semester 3 sobat. Adapun judul dari program yang mau saya posting ini sobat yaitu Program Sewa Mobil di C++ atau bisa juga disebut dengan program rental mobil dengan C++ sobat.

Program Sewa Mobil di C++ ini sobat menggunakan pernyataan NESTED IF (if bersarang) sobat ataupun biasa lebih dikenal dengan penyataan If dalam If, misalnya saja sobat jika pernyataan tersebut benar atau salah maka akan ada suatu pernyataan lagi dan jika setelah pernyataan itu terpenuhi masih ada pernyataan lagi maka akan dikerjakan lagi sampai perintah itu selesai sobat.

Adapun penjelasan dari coding dari program nilai mahasiswa dengan C++ ini adalah : 11. Pertama kita memasukkan 5 nilai mata kuliah dan sobat pun bisa menggantinya dengan beberapa mata kuliah sesuai dengan keinginan sobat loh, tapi sobat juga harus mengganti juga rumus jumlahnya sobat. 22. Penjumlahannya yaitu dengan memasukkan int jumlah = ak+agama+algo+sd+mt; itu int merupakan integer apabila kita mau menginputkan bilangan bulat dan apabila disitu float berarti untuk bilangan desimal. 33. Kemudian untul rata-ratanya sobat yaitu dengan rumus float rata2 = jumlah/5, itu maksudnya karena outputnya bilangan decimal, kemudian untuk jumlahnya dibagi 5 sobat. 44. Dan untuk penjelasan Ifnya(jika), yaitu apabila rata-ratanya diatas atau sama dengan 85 maka nilainya A dan seterusnya sesuai dengan coding/listing yang dimasukkan.
